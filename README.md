# BeautyCMS - 美容养生店小程序设计方案实现

这是一个基于 `uni-app` 前端和 `Node.js` 后端的美容养生店小程序全栈项目。

## 🏗️ 项目架构

```text
BeautyCMS/
├── frontend/          # 小程序前端 (uni-app)
│   ├── pages/         # 页面文件 (首页, 服务, 商城, 社区, 我的)
│   ├── static/        # 静态资源 (图标, 图片)
│   ├── components/    # 自定义组件
│   ├── api/           # 接口请求封装
│   └── manifest.json  # 小程序配置文件
├── backend/           # 管理后台后端 (Node.js + Express)
│   ├── controllers/   # 业务逻辑
│   ├── models/        # 数据库模型 (Sequelize/SQLite)
│   ├── routes/        # API 路由
│   ├── database/      # 数据库配置
│   └── index.js       # 入口文件
└── docs/              # 项目相关文档
```

## 🚀 核心功能实现

1. **用户系统**: 微信登录集成, 会员等级体系(普通/银卡/金卡/钻石), 积分系统。
2. **产品/服务系统**: 服务分类展示(面部护理/身体护理), 技师预约功能, 商品在线购买。
3. **营销系统**: 优惠券发放(新人券/满减券/折扣券), 限时秒杀活动, 积分兑换。
4. **管理后台**: 提供用户管理、订单管理、服务项目管理等核心接口。

## 🛠️ 技术栈

- **前端**: `uni-app` (Vue 3 + Vite)
- **后端**: `Node.js` + `Express`
- **数据库**: `SQLite` (开发阶段) / `MySQL` (生产建议)
- **ORM**: `Sequelize`

## 📦 快速开始

### 后端运行
1. 进入 `backend` 目录
2. 安装依赖: `npm install`
3. 启动服务: `npm run dev` (默认运行在 3000 端口)

### 前端运行
1. 进入 `frontend` 目录
2. 使用 HBuilderX 导入项目
3. 选择 "运行到小程序模拟器"

## 👤 会员等级规则
- **普通会员**: 注册即是
- **银卡会员**: 累计消费满1000元 (9.5折)
- **金卡会员**: 累计消费满5000元 (9折)
- **钻石会员**: 累计消费满20000元 (8.5折)
